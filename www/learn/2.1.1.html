Весьма важный частный случай дерева - бинарное дерево, в котором степень исхода в каждом узле не превышает 2. Если степень исхода в каждом узле равна 2 или нулю, то получается  полное бинарное дерево.

Множество строк, соответствующих листьям некоторого дерева, образуют  префиксный код этого дерева. Так, для дерева на рис. 2.1 б префиксный код имеет вид {00,010,011,10,11}.


Бинарные деревья на рис. 2.1 а, б представляют собой разные позиционные деревья.

 
 


Полное бинарное дерево выcomы h - это такое бинарное дерево, в котором каждый узел уровня h является листом и каждый узел уровня меньше h имеет непустые левое и правое поддеревья. Пример полного бинарного дерева высоты 4 приведен на рис. 2.2.

Почти полное бинарное дерево - это такое бинарное дерево, для которого выполняются следующие условия:

·            каждый лист в дереве имеет уровень hили h-1.

·            если узел дерева имеет правого преемника уровня h-1, то все его левые преемники, являющиеся листьями, также имеют уровень h-1.

Можно расширить представление почти полных бинарных деревьев с помощью массивов до общего представления бинарных деревьев. Для этого сначала необходимо определить почти полное бинарное дерево, которое содержит требуемое бинарное дерево, а затем представить с помощью массива это почти полное бинарное дерево, полученное путем расширения исходного бинарного дерева. Hа рис. 2.3б приведено почти полное бинарное дерево для бинарного дерева, изображенного на рис. 2.3а.

При представлении почти полного бинарного дерева, полученного преобразованием неполного бинарного дерева, массивом элемент массива выделяется для представления узла дерева независимо от того, будет он содержать узел дерева или нет (рис. 2.3в).

 
 


Следовательно,  необходимо пометить неиспользуемые элементы массива как пустые узлы дерева.  Это может быть выполнено одним из двух способов.

При применении первого способа требуется поместить специальное значение в те позиции массива, которые представляют пустые узлы дерева. Такое значение не должно соответствовать  никакому возможному допустимому значению, которое может храниться в узле дерева. Так, в дереве, узлами которого являются целые положительные числа, пустой узел можно отметить каким-либо отрицательным числом.

Второй способ предполагает наличие дополнительного массива того же размера, что и массив, используемый для представления бинарного дерева. Элементы этого массива должны содержать значение 1 в позициях, соответствующих действительным узлам дерева, и 0 в позициях, соответствующих пустым узлам дерева.

Представление бинарного дерева массивом рекомендуется использовать, когда известно,  что они почти полные,  поскольку  в  этом случае нет пустых узлов и нет необходимости в использовании дополнительного массива. Оно также дает хорошие результаты и для бинарных деревьев, которые незначительно отличаются от почти полных бинарных деревьев.
